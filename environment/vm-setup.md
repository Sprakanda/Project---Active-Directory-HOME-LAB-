#VM Setup

## Host Platform
- VirtualBox 7.0 on Windows 11 Home

## VM Used
- Windows Server 2022 (Domain Controller)
- Windows 10 pro (Client)
- Kali Linux 2025.2
- Splunk Enterprise (Ubuntu VM)

## Specs
- Each VM with 4-6 GB RAM, 25-80 GB disk
- Network: Host-only + NAT
